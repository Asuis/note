近日康奈尔大学搞了一个图像风格迁移的项目，据介绍这个项目解决了图像转化过程中的「畸变」问题。

代码基于 Torch，并已经在 Ubuntu 14.04 LTS 系统测试。

项目依赖：

    TorchMatlab 或 Octave

CUDA：

    CUDAcudnn

具体的使用方法参照 Github 的 README。
